 Habits Tracker

Live site: https://habits-trackers7877.netlify.app/

A simple and clean habits-tracker web app (client-side) to help you build, track, and maintain daily habits. This repository contains the client-side code for the project and a live deployment is available at the URL above.

Key features:

1|Create & manage habits — Add new habits with title, category, description, reminder time and image.

2|Track progress — Mark actions completed and view streaks and activity history.

3|Filter & search — Quickly find habits by category or search text.

4|User profile & avatars — Supports user name and email display (client-side mock profile support).

5|Responsive UI — Works on desktop and mobile with an accessible, minimal design.

Why this project:

6|This project is great for anyone who wants a lightweight, client-side habit tracker to prototype ideas or to use as a starting point for adding backend features like authentication, database syncing, and push notifications.

Demo:

Visit the live site to see the application in action: https://habits-trackers7877.netlify.app/
